II. Genèse de l'AP

Cette AP fait suite à l'AP sur la correction de bug de l'application web de GSB. Ici en gardant l'application web préexistante, nous allons effectuer des évolutions demandées pour GSB. Cette AP constituera l'un des projets obligatoires présentés lors de l'épreuve E5 du BTS sur machine. Pour finir, cette AP pour but de nous faire travailler en mode projet (en équipe), de consolider nos bases en modèle MVC ainsi que gérer l'habilitation de différents profils sur une application web.


III. Environnement de travail

Voici les différents logiciels utilisés lors de cette AP :
   - Visual Studio Code
   - WampServer
   - Git

IV. Description de l'AP

Tout d'abord avant de commencer à coder, nous avons reçu un cahier des charges. Ce cahier des charges contient l'ensemble des fonctionnalités à ajouté à l'application web avec une description détaillant les besoins et les personnes ayant accès à cette fonctionnalité. Pour ma part je m'occuper des déplacements chez les pharmaciens pour présenter des produits. Les délégués médicaux se déplacent dans les pharmacies chez les médecins. Pour chaque déplacement, on conservera la date, un commentaire, la pharmacie concernée et le produit présenté. Les délégués doivent pouvoir saisir les caractéristiques de leurs déplacements dans les pharmacies et le responsable de la comptabilité devra disposer d’une fonctionnalité lui permettant d’obtenir les déplacements en pharmacie saisies par un délégué médical de son choix, afin de procéder aux contrôles avant de valider les remboursements. En résumé, j'ai effectué l'ajout, la consultation, la modification et la suppression des déplacements chez les pharmaciens.
         Ensuite, nous avons ajouté selon la fonctionnalité des tables et données dans la base de données. Pour finir, les habilitations sont gérées dans l'application et au niveau la base de données grâce à la table fonctionnalité et profil_fonct selon le profil de l'utilisateur et nous avons également réutilisé Git pour la gestion de versions de l'application web.
