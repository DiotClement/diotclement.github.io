TP : Evolution d'un site Web pour intégrer les mentions légales

I. Compétences mises en oeuvre

Développer la présence en ligne de l'organisation :
   ▸ Participer à la valorisation de l’image de l’organisation sur les médias numériques en tenant compte du cadre juridique et des enjeux économiques
   ▸ Participer à l'évolution d'un site Web exploitant les données de l'organisation

II. Genèse du TP

Ce TP a pour but de nous faire découvrir le fonctionnement des mentions légales d'un site web ainsi que leurs implémentations. Pour mettre en pratique ce TP, des mentions légales ont été mises sur le site web linkretz.

IV. Description du TP

Tout d’abord, les mentions légales désignent les mentions obligatoires qui doivent apparaître sur tout support de communication. Les mentions légales d'un site internet servent à assurer une certaine transparence, à rassurer les internautes sur l'identité de celui qui émet les informations. La personne morale ou physique devient l'éditeur du site et est responsable de son contenu. Ce TP vise également à améliorer le site web linkretz afin d'y ajouter les mentions légales.
