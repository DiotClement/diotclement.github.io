II. Déroulement du stage

Ce stage se déroule lors de la deuxième année du BTS SIO et dure 1 mois et demi. Les dates du stage étaient les suivantes : du 3 janvier au 18 février 2022. Enfin mon stage s'est déroulé dans l'entreprise BIC Rasoirs située à Longueil-Sainte-Marie.


III. Environnement de travail

Voici les différents logiciels utilisés lors de ce stage :
   - Framework maison
   - Notepad++
   - PuTTy
   - SQL Server


IV. Description du stage

Durant ce stage, j'ai effectué trois missions orientées développement informatique et une mission orientée systèmes et réseaux :
   - Amélioration de la liste incendie de l'usine avec la classe PHP FPDF
   - Configuration et installation de pc Windows 10 dans l’usine
   - Amélioration de la saisie de packing list de l’usine en PHP
   - Ajout d'une fonctionnalité d'un module web de l'usine avec un Framework PHP
