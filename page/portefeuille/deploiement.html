II. Déroulement du TP

Ce tp a pour but de nous montrer la cohabitation développeur/administrateur réseaux ainsi que le déploiement d'un service web. En outre avec mon collègue Nathan PETROWICK de la section réseau, nous avons pu déployer un service Web et documenter son processus.


III. Environnement de travail

Voici les différents logiciels utilisés lors de ce TP:
   - Visual Studio Code
   - FileZilla
   - VirtualBox

IV. Description du TP

Tout d'abord avant de commencer ce tp, j'avais préalablement mon application web "Gestion des remboursements de frais" fonctionnelle avec quelques améliorations en plus et mon collègue avait quant à lui une machine virtuelle opérationnelle avec Apache, MariaDB, un interpréteur PHP, PHPMyAdmin, un serveur SSH, un serveur FTP et enfin HTTPS configuré pour sécuriser les accès sur le serveur Apache.
         Maintenant voilà comment s'est déroulé ce tp, avant de rejoindre mon collègue j'ai extrait la base de données de l'application web et en ne gardant uniquement les inserts de table de référence ainsi que l'application web. Ensuite nos professeurs nous ont donné la tâche de déployer ce site web ainsi que de créer un compte utilisateur pour Madame KINTZLER et de rendre à la fin un compte rendu de ce déploiement.
         Pour ce faire avec mon collègue, j'ai transféré via FTP le script de la base de données, le script des comptes utilisateurs ainsi que l'application web. Nous avons exécuté les scripts puis mon collègue a modifié certains paramètres de configuration de la machine virtuelle pour prendre en compte l'application web. Enfin une dernière tâche était de rendre la connexion à PuTTY avec un clé SSH mais malheureusement nous ne sommes pas parvenus à réaliser cette tâche. Cependant nous avons quand même mis dans le compte rendu comment effectuer cette tâche. Et voilà le site a été déployé et nos professeurs ont vérifié son fonctionnement. Pour finir, tout ce processeur fût rédigé dans un compte rendu.
